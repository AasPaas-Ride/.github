# AasPaas Ride

AasPaas Ride is building local mobility tools focused on simpler, more accessible ride experiences.

## About

We are working on products that make nearby travel and ride coordination easier, with a focus on practical user experience, reliable flows, and real-world utility.

## What we're building

- Rider-facing experiences for discovering and booking nearby rides
- Driver or partner workflows for managing ride activity
- Shared platform components for authentication, location, and trip operations

## Repositories

This organization currently brings together the core product work across its main repositories:
- Repo 1: core application/product repository
- Repo 2: supporting platform/admin/backend repository

## Focus

- Clean product architecture
- Fast iteration
- Real-world deployment readiness
- Local-first transportation use cases
